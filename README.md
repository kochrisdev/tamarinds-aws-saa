# AWS Solutions Architect Associate (SAA-C03) Learning Guide

This repository provides a **practical, exam-focused learning path** for AWS Solutions Architect Associate.

## 🎯 Goal
- Pass AWS SAA certification efficiently
- Understand real-world architecture
- Build hands-on skills

## 📚 What You’ll Learn
- Core AWS services
- Architecture patterns
- High availability & scalability
- Security & cost optimization

## 🧠 How to Use This Repo
1. Follow the study plan (`00-Study-Plan.md`)
2. Study one module per day
3. Do labs after each topic
4. Review cheat sheets before exam

## ⏱ Recommended Timeline
- 2–3 weeks (intensive)
- 4–6 weeks (part-time)
