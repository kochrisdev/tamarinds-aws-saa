# Serverless (Lambda)

## 🧠 What is Lambda?
Run code without managing servers.

---

## ⚙️ Key Features

- Event-driven
- Auto scaling
- Pay per use

---

## 🔥 Use Cases

- APIs (API Gateway)
- File processing (S3 trigger)
- Automation

---

## ⚠️ Limits

- Max 15 minutes execution
- Stateless

---

## 🧪 Architecture

S3 Upload → Lambda → DynamoDB

---

## 💡 Exam Tip

Use Lambda when:
- No server management needed
- Event-driven workload
