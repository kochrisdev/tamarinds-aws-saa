# EC2 (Elastic Compute Cloud)

## 🧠 What is EC2?
Virtual servers in the cloud.

---

## ⚙️ Key Concepts

### Instance Types
- General Purpose → t3
- Compute → c5
- Memory → r5

👉 Exam Tip:
Pick based on **workload type**

---

### Pricing Models

| Type | Use Case |
|------|--------|
| On-Demand | Short-term |
| Reserved | Long-term |
| Spot | Cheap but interruptible |

---

## 🔥 Storage

- EBS → persistent
- Instance Store → temporary

---

## 📊 Scaling

- Auto Scaling Group (ASG)
- Load Balancer required

---

## ⚠️ Exam Traps

- Instance failure → use **ASG**
- Need HA → use **Multi-AZ + ELB**

---

## 🧪 Real Scenario

> Web app with unpredictable traffic

Solution:
- EC2 + Auto Scaling
- ALB
- Multi-AZ
